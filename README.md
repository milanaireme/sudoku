# Sudoku Afterglow

A standalone keyboard-first Sudoku web app with larger cell text, five stages, a local stage leaderboard, and a frozen-digit mode for filling one number across the board.

## Run it

Open [index.html](C:\Projects\Trend research\sudoku-web\index.html) directly in a browser.

## Keyboard controls

- `Arrow keys` or `W A S D` move around the board.
- `1-9` fills the selected cell.
- `0`, `Backspace`, or `Delete` clears the selected cell.
- `Shift` + `1-9` freezes a digit.
- `Enter` or `Space` places the frozen digit into the selected cell.
- `P` clears the frozen digit.
- `N` toggles notes mode.
- `U` undoes the last move.
- `H` gives a hint.
- `C` checks the board.
- `G` starts a new puzzle.
- `B` returns focus to the board.
